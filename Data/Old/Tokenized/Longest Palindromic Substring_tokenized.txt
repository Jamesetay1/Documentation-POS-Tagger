<p> Given a string <code> s </code> , return <em> the longest palindromic substring </em> in <code> s </code> . </p> 
<p> </p> 
<p> Example 1 : </p> 
<pre> 
Input : s = " babad " 
Output : " bab " 
Note : " aba " is also a valid answer . 
</pre> 
<p> Example 2 : </p> 
<pre> 
Input : s = " cbbd " 
Output : " bb " 
</pre> 
<p> Example 3 : </p> 
<pre> 
Input : s = " a " 
Output : " a " 
</pre> 
<p> Example 4 : </p> 
<pre> 
Input : s = " ac " 
Output : " a " 
</pre> 
<p> </p> 
<p> Constraints : </p> 
<code> 1 < = s . length < = 1000 </code> <code> s </code> consist of only digits and English letters ( lower - case and / or upper - case )
