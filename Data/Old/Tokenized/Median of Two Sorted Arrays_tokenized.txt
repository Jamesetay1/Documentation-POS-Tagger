<p> Given two sorted arrays <code> nums1 </code> and <code> nums2 </code> of size <code> m </code> and <code> n </code> respectively , return the median of the two sorted arrays . </p> 
<p> Follow up : The overall run time complexity should be <code> O ( log ( m + n ) ) </code> . </p> 
<p> </p> 
<p> Example 1 : </p> 
<pre> Input : nums1 = [ 1 , 3 ] , nums2 = [ 2 ] Output : 2 . 00000 Explanation : merged array = [ 1 , 2 , 3 ] and median is 2 . </pre> 
<p> Example 2 : </p> 
<pre> Input : nums1 = [ 1 , 2 ] , nums2 = [ 3 , 4 ] Output : 2 . 50000 Explanation : merged array = [ 1 , 2 , 3 , 4 ] and median is ( 2 + 3 ) / 2 = 2 . 5 . </pre> 
<p> Example 3 : </p> 
<pre> Input : nums1 = [ 0 , 0 ] , nums2 = [ 0 , 0 ] Output : 0 . 00000 </pre> 
<p> Example 4 : </p> 
<pre> Input : nums1 = [ ] , nums2 = [ 1 ] Output : 1 . 00000 </pre> 
<p> Example 5 : </p> 
<pre> Input : nums1 = [ 2 ] , nums2 = [ ] Output : 2 . 00000 </pre> 
<p> </p> 
<p> Constraints : </p> 
<code> nums1 . length = = m </code> <code> nums2 . length = = n </code> <code> 0 < = m < = 1000 </code> <code> 0 < = n < = 1000 </code> <code> 1 < = m + n < = 2000 </code> <code> - 106 < = nums1 [ i ] , nums2 [ i ] < = 106 </code>
